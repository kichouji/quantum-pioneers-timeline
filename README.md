# Quantum Pioneers Timeline

量子力学の開拓者たちの生涯と主要な業績を視覚的に確認できるインタラクティブなタイムラインビューアです。

![スクリーンショット](./images/screenshot.png)

## Features

- 生年から没年までをタイムラインで表示
- 年代のスライダーで各物理学者の年齢を確認可能
- 主要な業績をマーカーで表示（ノーベル賞は金色で表示）

## Prerequisite

動作にはReact環境が必要です。
